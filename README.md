# Auto Process Chrome Extension

This extension is used to help the process of automating data from spreadsheets to target websites..

## Installation Method (For Users):
1. Click the green **Code** button above, then select **Download ZIP**.
2. Extract the downloaded ZIP file to your computer.
3. Open Google Chrome and go to the address: `chrome://extensions/`
4. Enable **Developer Mode** in the top right corner.
5. Click the **Load unpacked** button.
6. Select the extracted folder (the folder containing the `manifest.json` file).
7. The extension is ready to use!
